# Pacto - Teste Java

## Installation and Usage (Deploy)


Install Docker Containers

```bash
  docker-compose build
```

```bash
  docker-compose up
```

## Installation and Usage (Development)

### Backend
```bash
  cd ./backend
  
  mvn clean install

  mvn spring-boot:run
```

### FrontEnd
```bash
  cd ./frontend
  
  npm install

  npm run start
```
